# Privacy Policy

**Last updated:** 2026-03-17

## Summary

This plugin does not collect, store, or transmit any personal data to third parties.

## Data flow

All API calls made by this plugin go directly from your local machine to Fennoa's servers (`app.fennoa.com`) using your own API credentials. No data passes through any intermediate servers operated by the plugin author.

## Credentials

Your Fennoa API key and company code are stored locally in environment variables or a `.env` file on your own machine. They are never logged, transmitted, or stored anywhere outside your local environment.

## Third-party services

This plugin communicates exclusively with:
- **Fennoa** (`app.fennoa.com`) — your accounting data provider

Fennoa's own privacy policy applies to all data stored in your Fennoa account.
